# GeoLens Privacy Policy

Last updated: [March 2026]

## Overview

GeoLens is a browser extension for GeoGuessr that provides a stats overlay with country-level tracking and map visualization.

GeoLens does **not** collect, transmit, store, or sell any personal data.

## Data Storage

All statistics and game-related data are stored locally in the user's browser using IndexedDB.

No data is sent to external servers.

## Data Collection

GeoLens does not:

- Collect personally identifiable information
- Transmit browsing history
- Use analytics or tracking services
- Share data with third parties
- Store data outside the user's browser
- Execute any code from external sources

The extension runs only on `https://www.geoguessr.com/*` and operates entirely locally.
